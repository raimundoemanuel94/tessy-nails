# Tessy Nails

Sistema de agendamento e gestao para studio de manicure/pedicure.

## Stack

- Next.js App Router
- React + TypeScript
- Tailwind CSS + shadcn/ui
- Firebase Auth, Firestore e Messaging
- Stripe Checkout
- PWA via `next-pwa`

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
npm run seed:services
```
